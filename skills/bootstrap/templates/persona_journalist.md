# 페르소나 — 매체운영자·기자

## 폴더 구조

```
<vault>/
├── raw/                    # 원본 자료 (인터뷰 녹취·보도자료·1차 취재 메모)
├── _meta/                  # 일관성 카드·점검 표
├── 취재원/                  # 사례 노드 (who) — 인터뷰이·제보자
├── 이슈/                    # 이슈 노드 (topic) — 추적 중인 의제
├── 기사/                    # 산출물 노드 (kind) — 발행 기사·기획안
└── 캔버스/                  # 인물·이슈 관계도
```

## Properties 4축 + 매체 도메인 확장

```yaml
---
time: 2026-05
who: 김아무개          # 취재원 (실명 또는 코드명)
topic: 지방의회_AI도입
kind: 기획안

source: 김아무개
issue: 지방의회_AI도입
medium: 자체매체        # 자체매체·기고·외부청탁
status: 취재중          # 취재중·송고·발행·보류
deadline: 2026-06-10
tags: [취재/현장, 정치, 진행중]
---
```

## 어휘 매핑

| 변수 | 매체 값 |
|---|---|
| `{persona}` | 매체운영자 |
| `{who_label}` | 취재원·인터뷰이 |
| `{topic_label}` | 이슈·테마 |
| `{kind_examples}` | 기사·기획안·인터뷰메모·취재노트·이슈브리프 |
| `{folder_who}` | 취재원 |
| `{folder_topic}` | 이슈 |
| `{example_topic}` | 지방의회 AI 도입 |
| `{example_who}` | 김아무개 (○○시의회 정책지원관) |
