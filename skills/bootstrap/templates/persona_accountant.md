# 페르소나 — 회계사

## 폴더 구조

```
<vault>/
├── raw/                    # 원본 자료 (LLM 손대지 않음)
├── _meta/                  # 일관성 카드·점검 표·회고
│   ├── 일관성카드.md
│   └── 2차빌드_점검.md      # (2차 빌드 실행 시 자동 생성)
├── 클라이언트/              # 사례 노드 (who)
├── 회계기준/                # 기준 노드 (topic)
└── 캔버스/                  # json-canvas 시각 관계도
```

## Properties 4축 + 회계사 도메인 확장

```yaml
---
# 4축 (필수)
time: 2026-Q1        # 시점
who: (주)가나         # 주체 — 클라이언트
topic: 수익인식       # 주제 — 회계기준·세무이슈
kind: 감사메모        # 문서종류 — 감사메모·자문메모·신고서

# 회계사 도메인 확장
client: (주)가나
year: 2025
status: 진행중       # 진행중·완료·보류
auditor: 본인
standard: K-IFRS 1115
tags: [클라이언트/제조, 진행중]
---
```

## 어휘 매핑 (LLM 프롬프트 치환용)

| 변수 | 회계사 값 |
|---|---|
| `{persona}` | 회계사 |
| `{who_label}` | 클라이언트 |
| `{topic_label}` | 회계기준·세무이슈 |
| `{kind_examples}` | 감사메모·자문메모·신고서·검토조서 |
| `{folder_who}` | 클라이언트 |
| `{folder_topic}` | 회계기준 |
| `{example_topic}` | 수익인식 (K-IFRS 1115) |
| `{example_who}` | (주)가나 |
