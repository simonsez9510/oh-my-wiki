# demo/ — 회계사 페르소나 데모 vault

이 폴더 전체를 옵시디언 vault로 등록하면 Oh My Wiki 스킬을 *바로* 시연할 수 있습니다.

## 사용법

1. 옵시디언 실행 → `Open folder as vault` → 이 `demo/` 폴더 선택
2. Claude Code 실행 (이 디렉토리 또는 부모 디렉토리에서)
3. Claude Code 안에서:
   ```
   /oh-my-wiki:bootstrap
   ```
4. 직무 묻는 단계에서 **"회계사"** 선택
5. raw/ 폴더의 3개 자료가 1차 빌드 거쳐 위키 노트로 변환됨
6. 2차 빌드 점검 표 확인 후 "진행" 입력 → 일관성 카드 5규칙 적용
7. 옵시디언 그래프뷰에서 클러스터 형성 확인

## 들어있는 자료

```
demo/
├── raw/
│   ├── 01_감사메모_(주)가나_초안.md    (사례 자료 — 클라이언트)
│   ├── 02_회계기준_K-IFRS_1115_요약.md (기준 자료)
│   └── 03_클라이언트_(주)다라_5월미팅.md (사례 자료)
└── _meta/
    ├── 일관성카드.md                    (스킬이 박는 5규칙)
    └── 페르소나.md                      (회계사 어휘 매핑)
```

## 기대 결과

1차 빌드 후:
- `클라이언트/(주)가나.md` 생성 — `회계기준/수익인식 (K-IFRS 1115)` 위키링크 자동 형성
- `클라이언트/(주)다라.md` 생성
- `회계기준/수익인식 (K-IFRS 1115).md` 생성 — `(주)가나`로부터 백링크 받음
- 고스트 노트 2개 (`리스 1116`·`연결범위 1110`) — raw 자료가 없는 회계기준 참조

옵시디언 그래프뷰에서 *사례 → 기준* 한 방향 화살표 클러스터가 그려집니다.

> 실행 *전에* 결과물 모양만 먼저 보고 싶다면 [`_after_example/`](_after_example/README.md) 참고 — 1차+2차 빌드를 거친 3개 노트 샘플이 그대로 들어있습니다.
