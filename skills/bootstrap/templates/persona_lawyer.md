# 페르소나 — 변호사

## 폴더 구조

```
<vault>/
├── raw/                    # 원본 자료
├── _meta/                  # 일관성 카드·점검 표
├── 의뢰인/                  # 사례 노드 (who)
├── 판례/                    # 판례 노드 (topic)
├── 법령/                    # 법령 노드 (topic)
└── 캔버스/                  # 시각 관계도
```

## Properties 4축 + 변호사 도메인 확장

```yaml
---
time: 2026-03
who: (주)나라
topic: 부당해고
kind: 자문의견서

client: (주)나라
case_no: 2026-자문-0042
jurisdiction: 서울중앙지법
precedent: 대법원2024다12345
status: 진행중
attorney: 본인
tags: [의뢰인/법인, 노동, 진행중]
---
```

## 어휘 매핑

| 변수 | 변호사 값 |
|---|---|
| `{persona}` | 변호사 |
| `{who_label}` | 의뢰인 |
| `{topic_label}` | 쟁점·법령·판례 |
| `{kind_examples}` | 자문의견서·소송서면·검토보고서·메모 |
| `{folder_who}` | 의뢰인 |
| `{folder_topic}` | 판례·법령 |
| `{example_topic}` | 부당해고 (근로기준법 §23) |
| `{example_who}` | (주)나라 |
