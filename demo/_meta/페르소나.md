# 페르소나 — 회계사

이 vault는 **회계사** 직무용으로 셋업됐다.

## 어휘 매핑 (LLM 호출 시 치환 기준)

| 변수 | 회계사 값 |
|---|---|
| `{persona}` | 회계사 |
| `{who_label}` | 클라이언트 |
| `{topic_label}` | 회계기준·세무이슈 |
| `{kind_examples}` | 감사메모·자문메모·신고서·검토조서·회계기준요약 |
| `{folder_who}` | 클라이언트 |
| `{folder_topic}` | 회계기준 |

## 4축 Properties 형식

```yaml
---
time: 2026-Q1        # 시점
who: (주)가나         # 주체 — 클라이언트
topic: 수익인식       # 주제 — 회계기준·세무이슈
kind: 감사메모        # 문서종류

# 도메인 확장
client: (주)가나
year: 2025
status: 진행중       # 진행중·완료·보류
auditor: 본인
standard: K-IFRS 1115
tags: [클라이언트/제조, 진행중]
---
```

## 폴더 구조

- `raw/` — 원본 자료 (LLM 손대지 않음)
- `_meta/` — 일관성 카드·페르소나·점검 표·회고
- `클라이언트/` — 사례 노드 (who)
- `회계기준/` — 기준 노드 (topic)
- `캔버스/` — json-canvas 시각 관계도

## 주의 — 회사 보안 정책

클라이언트 식별 정보·금액·합의 조건 등 민감 정보는 **회사 PC 로컬 vault만**. 클라우드 동기화 금지.
