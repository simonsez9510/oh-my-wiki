# 페르소나 — 연구자

## 폴더 구조

```
<vault>/
├── raw/                    # 원본 자료 (논문 PDF·1차 자료·인터뷰 전사)
├── _meta/                  # 일관성 카드·점검 표
├── 프로젝트/                # 사례 노드 (who) — 진행 중 연구 프로젝트
├── 개념/                    # 개념·이론 노드 (topic)
├── 출판/                    # 산출물 노드 (kind) — 논문·보고서·발표
└── 캔버스/                  # 개념지도·인용 네트워크
```

## Properties 4축 + 연구자 도메인 확장

```yaml
---
time: 2026-H1
who: AI거버넌스_연구
topic: 알고리즘책임
kind: 정책보고서

project: AI거버넌스_연구
method: 사례연구          # 사례연구·문헌고찰·실증·정책분석
funder: 자체              # 자체·국책·기업·외부재단
output: 정책보고서        # 논문·보고서·정책브리프·발표
status: 진행중
tags: [프로젝트/정책, AI, 진행중]
---
```

## 어휘 매핑

| 변수 | 연구자 값 |
|---|---|
| `{persona}` | 연구자 |
| `{who_label}` | 프로젝트 |
| `{topic_label}` | 개념·이론·방법론 |
| `{kind_examples}` | 논문·보고서·정책브리프·발표·메모·개념정리 |
| `{folder_who}` | 프로젝트 |
| `{folder_topic}` | 개념 |
| `{example_topic}` | 알고리즘 책임 |
| `{example_who}` | AI거버넌스_연구 (2026 H1) |
